# Per-claim checklist

- [ ] Exact claim text recorded
- [ ] Product/deployment/region/version scope recorded
- [ ] Evidence source and section recorded
- [ ] Evidence is current
- [ ] Exceptions documented
- [ ] Security owner approved technical accuracy
- [ ] Compliance/Privacy/Legal reviewed when relevant
- [ ] Customer-sharing classification set
- [ ] NDA requirement recorded
- [ ] Review/expiration date assigned
- [ ] RFP variants mapped to the canonical answer
- [ ] Agent-safe caveat added

# High-risk claim types

The following always require accountable-owner approval:

- Certifications and regulatory compliance
- Encryption and key management
- Availability, SLA, RTO, and RPO
- Data residency and deletion
- Incident notification
- Penetration testing and vulnerability remediation
- Customer-managed deployment and shared responsibility
- Contractual exceptions
